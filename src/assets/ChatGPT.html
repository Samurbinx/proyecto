Para sumar los puntos de todas las revisiones (reviews), puedes crear una función en Angular que recorra todas las revisiones y sume sus puntos. Aquí tienes un ejemplo de cómo podrías implementarlo en un componente de Angular:

typescript
import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  reviews: any[] = [
    { points: 5 },
    { points: 3 },
    { points: 4 }
    // Aquí irían más revisiones si las tienes
  ];

  sumPoints(): number {
    let totalPoints = 0;
    if (this.reviews && this.reviews.length > 0) {
      this.reviews.forEach(review => {
        totalPoints += review.points;
      });
    }
    return totalPoints;
  }
}

Luego, en tu plantilla (HTML), puedes llamar a esta función sumPoints() para mostrar la suma de los puntos de todas las revisiones:

html
<p>Total de puntos: {{ sumPoints() }}</p>

Este código recorrerá el arreglo de revisiones (reviews), sumará los puntos de cada revisión y devolverá el total. Si no hay revisiones o si el arreglo está vacío, devolverá 0.

Para realizar la validación de un formulario en Angular, puedes aprovechar las capacidades que ofrece Angular Forms. A continuación, te mostraré un ejemplo básico de cómo podrías hacerlo:

Primero, en tu archivo de componente (por ejemplo, app.component.ts), deberías importar las clases necesarias para trabajar con formularios y luego definir tu formulario utilizando el módulo FormGroup y FormControl, así como cualquier validación que desees aplicar.

typescript
import { Component } from '@angular/core';
import { FormGroup, FormControl, Validators } from '@angular/forms';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html',
  styleUrls: ['./app.component.css']
})
export class AppComponent {
  formulario: FormGroup;

  constructor() {
    this.formulario = new FormGroup({
      nombre: new FormControl('', Validators.required),
      email: new FormControl('', [Validators.required, Validators.email]),
      edad: new FormControl('', Validators.required)
    });
  }

  submitFormulario() {
    if (this.formulario.valid) {
      // Aquí puedes enviar el formulario o realizar otras acciones
      console.log('Formulario válido. Datos enviados:', this.formulario.value);
    } else {
      // Si el formulario no es válido, puedes mostrar un mensaje de error o realizar otra acción
      console.error('Formulario inválido. Por favor, corrija los errores.');
    }
  }
}

Luego, en tu archivo de plantilla HTML (por ejemplo, app.component.html), puedes enlazar los controles de formulario y mostrar mensajes de error si es necesario.

html
<form [formGroup]="formulario" (ngSubmit)="submitFormulario()">
  <div>
    <label for="nombre">Nombre:</label>
    <input type="text" id="nombre" formControlName="nombre">
    <div *ngIf="formulario.get('nombre').invalid && formulario.get('nombre').touched">
      <div *ngIf="formulario.get('nombre').errors.required">Nombre es requerido.</div>
    </div>
  </div>
  <div>
    <label for="email">Email:</label>
    <input type="email" id="email" formControlName="email">
    <div *ngIf="formulario.get('email').invalid && formulario.get('email').touched">
      <div *ngIf="formulario.get('email').errors.required">Email es requerido.</div>
      <div *ngIf="formulario.get('email').errors.email">Email no es válido.</div>
    </div>
  </div>
  <div>
    <label for="edad">Edad:</label>
    <input type="number" id="edad" formControlName="edad">
    <div *ngIf="formulario.get('edad').invalid && formulario.get('edad').touched">
      <div *ngIf="formulario.get('edad').errors.required">Edad es requerida.</div>
    </div>
  </div>
  <button type="submit" [disabled]="formulario.invalid">Enviar</button>
</form>

En este ejemplo:

  • Se define un formulario con tres campos: nombre, email y edad.
  • Cada campo tiene una validación, como "required" para campos obligatorios y "email" para el campo de correo electrónico.
  • El método submitFormulario() se llama cuando se envía el formulario. Este método verifica si el formulario es válido antes de realizar acciones adicionales.
  • Se muestra un mensaje de error debajo de cada campo del formulario si es inválido y se ha tocado (es decir, el usuario ha interactuado con él).

import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';
import { Observable } from 'rxjs';
import { PlaceModel } from './place.model';

@Injectable({
  providedIn: 'root'
})
export class PlaceService {
  private URL_API = '...'; // Coloca aquí la URL de tu API

  constructor(private _http: HttpClient) {}

  getPlace(id: number): Observable<PlaceModel> {
    return this._http.get<PlaceModel>(`${this.URL_API}/places/${id}`);
  }

  editPlacePoints(id: number, pts: number): Observable<PlaceModel> {
    // Realiza la solicitud HTTP PUT para actualizar solo los puntos del lugar
    return this._http.put<PlaceModel>(`${this.URL_API}/places/${id}`, { points: pts });
